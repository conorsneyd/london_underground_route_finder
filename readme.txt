London Underground Route Finder
_________________________



About:

A Flask-based web application which finds the fastest route between any two London Underground stations by traversing a graph of vertices using breadth-first search, then converts the route into user-friendly text.

Created by Conor Sneyd (https://github.com/conorsneyd)



Requirements:

Python 3.12, Flask



How to run:

Open app.py in Python, then open http://localhost:5000/ in browser and follow the instructions on screen.



Data:

Information on train routes taken from https://tfl.gov.uk/
Information on journey time between stations taken from http://www.london-tubemap.com/assets/tubemap_journey-times_2.pdf
